# Rallye

App mobile para controle transparente de aulas entre professor e aluno de esportes de raquete (padel, tênis, squash, badminton).

## Setup

### Requisitos

- [FVM](https://fvm.app) — gerenciador de versões do Flutter
- [Supabase CLI](https://supabase.com/docs/guides/cli)

### Instalação

```bash
fvm install
fvm flutter pub get
```

### Variáveis de ambiente

Copiar `.env.example` para `.env` e preencher as credenciais:

```bash
cp .env.example .env
```
